Blank-page syndrome, gone. Describe what you want and Fluso produces it — documents, presentations, spreadsheets, images, diagrams, and web pages. All professionally formatted, ready to use.

## What you can create

| Type | Examples |
|---|---|
| **Documents** | Reports, proposals, memos, letters, manuals, newsletters |
| **Presentations** | Pitch decks, board decks, training, conference talks, webinars |
| **Spreadsheets** | Budgets, trackers, analyses, dashboards with formulas and charts |
| **Images** | Logos, social graphics, hero images, mockups, icons, banners |
| **Diagrams** | Flowcharts, architecture, process maps, org charts, mind maps |
| **Web pages** | Landing pages, dashboards, full React applications |

## Documents (PDF & Word)

### Reports and proposals

> *"Write a project proposal for the Acme partnership — executive summary, scope, timeline, budget, team. Professional formatting with a table of contents."*

> *"Create a Q2 performance report with sections for revenue, product metrics, team updates, and Q3 priorities. Include data tables and charts."*

> *"Write a client memo about the new pricing changes taking effect next month. Formal but friendly tone."*

### What Fluso handles

- **Formatting** — headings, fonts, spacing, margins.
- **Structure** — TOC, page numbers, sections, appendices.
- **Tables** — data organised and formatted.
- **Images** — charts, diagrams, and visuals placed appropriately.
- **Branding** — consistent colours, fonts, and style.

### Common formats

| Ask for | Fluso creates |
|---|---|
| Project proposal | Executive summary, problem, solution, timeline, budget, team |
| Board report | Financial highlights, KPIs, wins, challenges, recommendations |
| Client memo | Subject, context, details, next steps, professional formatting |
| Meeting agenda | Objectives, topics, time allocations, prep materials |
| Onboarding guide | Welcome, setup, resources, contacts, FAQ |
| Case study | Challenge, solution, results, quotes, metrics |

### Working with existing documents

> *"Turn these meeting notes into a formatted report."*

> *"Merge these three PDFs into one document."*

> *"Extract all tables from this scanned PDF."*

> *"Add a watermark and page numbers."*

> *"Create a budget spreadsheet with Q2 projections by department — actuals, forecast, and variance columns. Add formulas and charts."*

## Tips

<Tip>
  **Be specific about audience, tone, and format.** "Create a 5-slide board presentation about Q3 metrics — data-heavy, formal tone, include recommendations" produces far better output than "make a presentation."
</Tip>

<Tip>
  **Iterate.** The first version is great. "Add more detail about the competition" or "Make the design more modern" gets you to perfect.
</Tip>

<Tip>
  **Use your brand.** Tell Fluso "Use our brand colours: #7C3AED primary, #1E1B4B dark" and it stays consistent across documents, decks, and images.
</Tip>

<Tip>
  **Personalise external-facing content.** Fluso's output is excellent but adds value when you put your voice on top — especially pitch decks and client proposals.
</Tip>
